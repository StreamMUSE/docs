    模块文档索引¶

    English | 中文

    本目录包含了模型训练框架所有模块的详细文档说明。

    核心模块¶

    Model Module¶

    模型模块是深度学习训练框架的核心抽象层，基于 PyTorch Lightning 构建。

    Network Module¶

    网络模块专注于神经网络架构的定义。

    Dataset Module¶

    数据集模块专注于数据加载、预处理和模型输入的统一管理。

    工具模块¶

    Callback Module¶

    回调模块专注于 PyTorch Lightning 训练过程中的回调功能配置。

    DataModule Module¶

    数据模块专注于数据加载和准备，与训练逻辑解耦。

    Logger Module¶

    日志器模块专注于实验日志记录和管理。

    LR Scheduler Module¶

    学习率调度器模块专注于训练过程中的学习率调整策略。

    Optimizer Module¶

    优化器模块专注于模型参数优化的配置和管理。

    Trainer Module¶

    训练器模块专注于 PyTorch Lightning Trainer 的配置和管理。

    Utils Module¶

    工具模块提供通用的配置基础类和工具函数。

    Project Module¶

    项目模块专注于整个训练项目的组织和执行。

    Metric Module¶

    指标模块专注于指标计算、记录和管理。
